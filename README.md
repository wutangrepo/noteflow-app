# üìù NoteFlow

> **Product Vision:** To create a lightweight, distraction-free collaboration space where distributed student teams can sync their daily progress instantly.

## üìñ About The Project
**NoteFlow** is a web-based "Digital Standup Board" developed as part of our **Team Project of IT** course. 

The goal of this project is to simulate a real-world Scrum development cycle. We focus on enabling distributed teams to post quick status updates without the complexity of heavy project management tools.

### üöÄ MVP Features (v1.0 Release)
For our initial Minimum Viable Product, we are delivering:
*   **Post Notes:** Create distraction-free status updates.
*   **Delete Notes:** Remove outdated tasks easily.
*   **Data Persistence:** Browser LocalStorage integration (Data survives page refreshes).
*   **Team Identification:** Simple author tagging to know "Who wrote what."

## üíª Getting Started
### Prerequisites
*   A modern web browser (Chrome, Firefox, Edge, Safari).
*   VS Code (recommended for editing).

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/YOUR_USERNAME/noteflow-app.git
    ```
2.  **Navigate to the project folder**
    ```bash
    cd noteflow-app
    ```
3.  **Launch the App**
    *   Simply open the `index.html` file in your browser.
    *   *Or use Live Server in VS Code.*
